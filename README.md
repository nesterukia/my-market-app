# Витрина интернет-магазина с использованием Spring Boot

Полнофункциональное веб-приложение интернет-магазина с использованием Spring Boot, Thymeleaf, JPA и PostgreSQL.

### Основной функционал

- **Каталог товаров** - просмотр товаров с поиском, сортировкой (по алфавиту/цене) и пагинацией

- **Корзина** - добавление/удаление товаров, изменение количества, подсчет общей стоимости

- **Заказы** - создание заказов из корзины, просмотр истории заказов

- **Case-insensitive поиск** - по названию и описанию товаров

### Веб-интерфейс (Thymeleaf)

- `GET /` или `/items` - каталог товаров (поиск, сортировка, пагинация)

- `GET /items/{id}` - детальная страница товара

- `GET /cart/items` - содержимое корзины

- `GET /orders` - список всех заказов

- `GET /orders/{id}` - детали заказа

- `POST /buy` - оформить заказ из корзины

### Запуск приложения

1. Клонируйте репозиторий
2. Установите Docker и Liquibase
3. По необходимости отредактируйте файл `application.yml`
4. По необходимости отредактируйте файл `docker-compose.yml`
5. Выдайте права на исполнение и запустите скрипт:

```bash
   chmod +x start.sh && ./start.sh
   ```

### Остановка приложения

Для остановки приложения необходимо выдать права на исполнение и запустить скрипт:
```bash
   chmod +x stop.sh && ./stop.sh
   ```
