# 🚀 Мультимодульный прооект "Витрина интернет-магазина" + "Сервис платежей" на Spring WebFlux + PostgreSQL + Redis

## 💰 Сервис платежей

### Основной функционал

- **Получение баланса на счёте** - (фиксированный баланс - 1000 рублей) — если баланс меньше суммы цены товаров в корзине, то кнопка оформления заказа основного приложения недоступна;
- **Осуществление платежа** - (при нажатии на кнопку оформления заказа происходит запрос на вычитание суммы заказа из баланса) — если оно даёт отрицательный результат, то считается, что оплата заказа не прошла и он не оформляется, и выдаётся соответствующая ошибка.

### Веб-интерфейс (RESTful API)

Веб-интерфейс описан в [openapi.yaml](payment-service/src/main/resources/openapi.yaml)

## 🛒 Витрина интернет-магазина с использованием Spring Boot

### Основной функционал

- **Каталог товаров** - просмотр товаров с поиском, сортировкой (по алфавиту/цене) и пагинацией

- **Корзина** - добавление/удаление товаров, изменение количества, подсчет общей стоимости

- **Заказы** - создание заказов из корзины, просмотр истории заказов

- **Case-insensitive поиск** - по названию и описанию товаров

### Веб-интерфейс (Thymeleaf)

- `GET /` или `/items` - каталог товаров (поиск, сортировка, пагинация)

- `GET /items/{id}` - детальная страница товара

- `GET /cart/items` - содержимое корзины

- `GET /orders` - список всех заказов

- `GET /orders/{id}` - детали заказа

- `POST /buy` - оформить заказ из корзины

### Запуск приложения

1. Клонируйте репозиторий
2. Установите Docker и Liquibase
3. Задайте необходимые параметры в файле `.env`
4. По необходимости отредактируйте файл `application.yml`
5. По необходимости отредактируйте файл `docker-compose.yml`
6. Выдайте права на исполнение и запустите скрипт:

```bash
   chmod +x start.sh && ./start.sh
   ```

### Остановка приложения

Для остановки приложения необходимо выдать права на исполнение и запустить скрипт:
```bash
   chmod +x stop.sh && ./stop.sh
   ```
